# Quick Document Summary

Immediate conversational summaries. Output to conversation only — never modify the source file.

**Lead with insights, not descriptions.** "This document discusses..." is wrong. Start with the core finding or recommendation.

Scale depth to document length — 2-3 sentences for short docs, structured sections for long ones.
