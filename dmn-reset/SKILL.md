# DMN Reset

5-minute protocol to interrupt Default Mode Network racing thoughts. Doesn't require a calm mind to start — it creates calm through execution.

## Minute 1: Breathing Reset
**4-4-6**: Inhale 4s → Hold 4s → Exhale 6s → ~5 cycles. Extended exhale activates parasympathetic. If mind wanders, return to counting.

## Minute 2: Physical Anchoring
Recall a confident moment, amplify sensory details. Create physical trigger (fist, thumb press) while intensifying feeling. Repeat 3-4 times. Future use: fire anchor before reactive situations.

## Minute 3: Cognitive Reframing
Identify one limiting thought → Challenge ("Is this completely true? Always?") → Find contradicting evidence → Reframe:
- "I have no time" → "I'm prioritizing appropriately right now"
- "Nothing works" → "Current approaches haven't; other options exist"

## Minute 4: Gratitude Shift
3 specific appreciations (not "my family" but "Zoey's laugh this morning"). Feel each physically. Include one challenge that taught something.

## Minute 5: Autosuggestion
Repeat slowly, synchronized with breathing:
- **Default**: "I am choosing healthy choices toward my goals"
- **Firefighting**: "I respond with calm systematic thinking"
- **Recovery**: "I am actively rebuilding my energy and curiosity"
