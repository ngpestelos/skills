Transform vague existential distress into structured, actionable inquiry. Create audit frameworks with journaling prompts, not advice or philosophical lectures.

### Step 1: Identify the Core Question
Extract: What is absent? (meaning, joy, peace, direction). What temporal orientation? (past loss, present absence, future uncertainty). What life domain? (work, family, self, all).

### Step 2: Design Multi-Angle Sections
Build sections approaching the question from complementary angles:

| Angle | Focus |
|-------|-------|
| Future | Vision/Projection |
| Past | Diagnosis/Analysis |
| Counterfactual | Prerequisite/Condition |
| Relational | Legacy/Impact |
| Existential | Direct Confrontation |
| Synthesis | Definition |
| Application | Decision Implication |

Always end with a synthesis prompt ("Based on everything above, complete: 'Peace, for me, requires: ___'") and a decision-grounding prompt connecting the abstract to a concrete current choice.
