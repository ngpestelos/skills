# Reaction Gap Practice

"Between stimulus and response there is a space." — Frankl. This skill widens that space.

## Name-Pause-Choose Protocol

### 1. Name

Label the emotion: "I notice I'm feeling frustrated" / "Anger is arising"

Affect labeling shifts processing from limbic to prefrontal cortex, reducing emotional intensity.

### 2. Pause

| Situation | Technique |
|-----------|-----------|
| Conversation | One deliberate breath before speaking |
| Email/Message | Save as draft, return in 10+ minutes |
| Important decision | "Sleep on it" rule |
| Heated moment | "Let me think about that" |

### 3. Choose

- What would the person I want to become do here?
- How will I feel about this response tomorrow?

## Implementation Intentions

```
WHEN I feel [emotion]
IN situation [context]
I WILL [specific response]
```

- WHEN defensive IN code review → "That's interesting, can you explain your reasoning?"
- WHEN frustrated IN parenting → get on their level, lower voice
- WHEN anxious IN deadline pressure → list top 3 priorities before acting

## Rules

- Never say "don't feel that way" — emotions arise involuntarily, reactions are trainable
- Suppressing emotions backfires; choosing responses works — accept the feeling, control the action
