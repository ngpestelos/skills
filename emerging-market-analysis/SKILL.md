# Emerging Market Analysis Framework

## Framework 1: Governance Trauma Pricing Cycle

```
GOVERNANCE SHOCK → Initial Crash → Sentiment Overshoot ("Uninvestable")
→ Kitchen-Sink Pricing (all bad news public) → Bad News Buffer (diminishing impact)
→ Recovery Begins → Mean Reversion
```

| Phase | Signs |
|-------|-------|
| Overshoot | Consensus "uninvestable", institutional underweight |
| Kitchen-Sink | Details fully public, media saturated, "old news" |
| Bad News Buffer | Negative headlines have diminishing price impact |
| Recovery | Technical breakout, volume surge, regional rotation |

Don't buy during Initial Crash. Watch for Kitchen-Sink. Position during Bad News Buffer.

## Framework 2: Central Bank Signaling Constraint

| Action | Direct Effect | Signal Effect |
|--------|---------------|---------------|
| 50bp cut | More stimulus | "We're desperate" |
| 25bp cut | Moderate stimulus | "Measured, in control" |
| Off-cycle cut | Timely response | "Emergency, panic" |
| Scheduled cut | Predictable | "Competent, planned" |

**BSP pattern**: Prefers 25bp moves, prioritizes signaling over stimulus magnitude.

## Framework 3: Exchange Rate-Inflation Feedback

```
Rate Cuts → Yield Differential Narrows → Capital Outflows → Currency Weakens
→ Import Prices Rise → Inflation Increases → Forces Rate Hikes (undoing cuts)
```

| Metric | What It Tells You |
|--------|-------------------|
| Yield spread vs Fed | Room for independent policy |
| Import dependence | Sensitivity to currency moves |
| Current account | External financing needs |
| FX reserves | Ability to defend currency |

## Framework 4: Valuation Floor Identification

| Signal | Threshold |
|--------|-----------|
| P/E vs historical mean | -2 SD |
| P/E vs regional peers | >20% discount |
| Price vs multi-year range | Near lows |
| Sentiment | Extreme pessimism ("uninvestable") |

## Framework 5: Capital Rotation Timing

| Indicator | Leader → Laggard Signal |
|-----------|-------------------------|
| DXY | Falling (EM tailwind) |
| Gold | Rising (dollar alternative) |
| EM vs DM relative | EM outperforming |
| Small vs large cap | Small caps leading |
| Value vs growth | Value outperforming |
