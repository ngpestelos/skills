# Capture What Resonates

Guide knowledge capture decisions using intuitive resonance rather than analytical evaluation.

## The Resonance Filter

**Capture when you notice**: Curiosity spark ("I want to know more"), wonder ("I never thought about it that way"), excitement ("This could change how I approach X"), recognition ("This connects to something I've been thinking about"), surprise ("This contradicts what I assumed"), or it helps answer one of your Twelve Favorite Problems.

**Don't capture when you feel**: Obligation ("I should probably save this"), FOMO ("Everyone thinks this is important"), completism ("I need the whole article"), hoarding ("This might be useful someday").

**Don't capture at all**: Sensitive info (use password manager), specialized formats (use dedicated apps), very large files (use cloud storage), collaborative docs (use shared platforms).

**The Ken Burns Principle**: Capture only what naturally resonates after 1-2 hours. Good capture involves "forgetting as much as possible" — letting non-essential information fade so truly resonant insights remain.

## Key Rules

- Trust gut response in 5 seconds — analysis paralysis defeats the purpose
- Capture resonant passages, not entire articles — future processing time becomes prohibitive
- Treat personal realizations as capture-worthy — your own insights are often most valuable
- Capture first, organize through use — don't create elaborate structure before content exists
- When uncertain about placement, default to your Resources folder
