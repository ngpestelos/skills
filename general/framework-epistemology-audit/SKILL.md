## Core Principles

1. **Lenses are not laws** — A lens reveals features; a law constrains reality. Name which one.
2. **Every compression distorts** — Cross-domain mappings compress away domain-specific details. Name the distortion cost.
3. **Credit prior art** — Check Shannon, Wiener, Prigogine, Mandelbrot, Dawkins, Ashby before accepting novelty claims.

## Three Pressure Tests

### Test 1: Falsifiability
**What would count as evidence against this?**

| Result | Classification |
|--------|---------------|
| Specific counterexamples would violate it | Falsifiable — genuine constraint |
| Any outcome retroactively explained | Unfalsifiable — teleological |
| Circular definition | Tautological |

### Test 2: Independence
**Can each "law" be derived from the others?**
If Law A → Law B → Law C → Law A, they're one claim with multiple descriptions. Compare: Thermodynamics has four genuinely independent laws.

### Test 3: Predictive Specificity
| Result | Classification |
|--------|---------------|
| Quantitative prediction | Law-grade specificity |
| Directional tendency | Observation, not law |
| Post-hoc explanation | Narrative, not prediction |

## Pattern Verification Hierarchy

| Level | Name | Definition | Example |
|-------|------|-----------|---------|
| **1** | Structural isomorphism | Same math, same mechanism | Exponential decay in isotopes and capacitors |
| **2** | Shared mechanism | Same process, different substrate | Feedback loops in thermostats, ecosystems, markets |
| **3** | Analogical pattern | Similar shape, different mechanism | "Viral" spread of ideas vs epidemiology |
| **4** | Metaphorical resemblance | Poetic similarity, no structural basis | "The economy is sick" |

**Classification**: Identical math? → L1. Same causal mechanism? → L2. Only shape/narrative similarity? → L3-4.
**Warning**: Most pop-intellectual frameworks claim Level 1 while operating at Level 3-4.

## Systems Hierarchy Check

```
Level 1: Physical laws (thermodynamics, Shannon, Landauer)
Level 2: Systems principles (feedback, emergence, coherence)
Level 3: Applied sciences (markets, biology, psychology)
Level 4: Daily life applications
```
**Common error**: Taking Level 2 observations and labeling them Level 1 laws.

## Key Rules

- Name the distortion cost of every cross-domain mapping
- Steelman the framework before critiquing
- Never dismiss patterns just because packaging is wrong — the pattern may be real
- Never accept "everything is X" without decomposing X into deeper assumptions
- Always reconstruct what's salvageable — audit is not purely destructive
