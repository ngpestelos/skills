# Pre-Learning Prep

Systematic knowledge review before engaging with new material — activating existing mental models to enhance retention and enable cross-domain synthesis.

## The Two-Stage Connection Process

**Stage 1: Early Connections** (before/during initial learning) — Map source structure onto existing knowledge: "How does this relate to what I already know? What frameworks apply? Where does this fit?"

**Stage 2: Later Connections** (after initial learning) — Cross-disciplinary synthesis: "What patterns from other domains apply? How does this connect to unrelated topics?" Pre-primed connections are stronger than hoping for serendipity.

## The Pre-Learning Checklist (5-15 min)

### Step 1: Search Existing Knowledge
- Check your knowledge base for existing notes on the topic
- Review related project notes and reading highlights

### Step 2: Activate Mental Models
1. **"What do I already know?"** — Write 3-5 key concepts you already hold
2. **"What frameworks apply?"** — List mental models, note cross-domain patterns
3. **"Where are my gaps?"** — What questions and confusion exist?

### Step 3: Set Learning Intentions
- What specific questions do I want answered?
- What connections am I hoping to make?
- How will I know if this learning was valuable?

### Step 4: Prime Connection Candidates
Identify 2-3 topics that might connect: related notes, opposing frameworks, different domain applications.

## Quick Version (2 min)

1. **One search** for the main topic in your knowledge base
2. **Three questions**: What do I know? What framework applies? What's my main question?
3. **One connection candidate**: What other topic might this relate to?

## Integration with Reading Workflow

| Phase | Actions |
|-------|---------|
| Before | Set up note template, note initial questions |
| During | Mark resonant passages, note connections to primed topics, flag synthesis opportunities |
| After | Complete summary layers, execute Stage 2 connections, update knowledge base |
